## Antes de empezar — Conceptos base

Si nunca tocaste HTML y CSS, te recomendamos revisar estos recursos antes de los ejercicios:

- [MDN — Introducción a HTML](https://developer.mozilla.org/es/docs/Learn/HTML/Introduction_to_HTML)
- [MDN — Primeros pasos en CSS](https://developer.mozilla.org/es/docs/Learn/CSS/First_steps)
- [The Odin Project — Foundations](https://www.theodinproject.com/paths/foundations)

No es obligatorio completarlos primero, pero ayudan a tener contexto antes de cada ejercicio.

## Herramientas recomendadas

- Editor: VS Code con extensión "Live Server"
- Validador HTML: https://validator.w3.org/
- Can I Use: https://caniuse.com/
- Devtools del navegador (F12)

## Cómo navegar los ejercicios

Cada carpeta de ejercicio contiene:

```
ejercicio-XX_nombre/
├── SPEC.md        ← Lee esto primero. Son los requerimientos.
├── REFLEXION.md   ← Completá esto DESPUÉS de terminar el código.
└── solucion/      ← Acá va todo tu código.
    └── (vacío — vos lo llenás)
```

> 💡 **Tip:** No mires soluciones de otros estudiantes antes de intentar el ejercicio. El proceso de pensar y equivocarse es parte del aprendizaje.
